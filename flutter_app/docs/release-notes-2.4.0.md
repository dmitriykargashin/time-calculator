### Highlights in this release (for reference / the longer store description)

- **Customisable keypad keys (NEW):** choose exactly which time-unit keys the
  keypad shows, in Settings → Keypad keys. One-tap presets — **Standard**,
  **Stopwatch** (Min·Sec·Msec), **Media** (Hour·Min·Sec·Msec for video/audio),
  **Hours & minutes**, **Calendar**, **Everything** — plus per-unit chips and a
  **live whole-keypad preview** (portrait + landscape carousel). Fewer keys give
  a cleaner, more compact keypad.
- **Result action menu (NEW):** a single tap on the result opens Copy / Change
  result format / Rate calculator / Share; a long press starts native text
  selection.
- **Calculation history:** the recent results are stored locally; per-record
  note, date, delete, and "clear history". Tapping an entry reloads the
  expression *and* restores the result format it was saved with.
- **Rate ("Per") calculator** redesigned for clarity (amount per time interval).
- **More result formats:** milliseconds and Hour·Minute·Second·Msec.
- **Smoother UI:** the expression and result scale almost continuously as the
  display/keypad split is dragged.
- **Polish & fixes:** gentle "add a time unit" hint for unitless input, fixed
  dark-theme dividers, a single copy confirmation (Android uses its own system
  one), and an accessible, labelled result button.
