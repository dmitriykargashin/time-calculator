### Highlights in this release (for reference / the longer store description)

- **Calculation history (F6):** the recent results are stored locally; per-record
  note, date, delete, and "clear history". Tapping an entry reloads the
  expression *and* restores the result format it was saved with. On by default.
- **Result action menu:** a single tap on the result opens Copy / Change result
  format / Rate calculator / Share; a long press starts native text selection.
- **Rate ("Per") calculator** redesigned for clarity (amount per time interval).
- **More result formats:** milliseconds and Hour·Minute·Second·Msec; optional
  **Msec** keypad key (replaces Year), Msec is the default.
- **Smoother UI:** the expression and result now scale almost continuously as the
  display/keypad split is dragged (matching the keys).
- **Polish & fixes:** gentle "add a time unit" hint for unitless input, fixed
  dark-theme dividers, single copy confirmation (Android uses its own system
  one), and an accessible, labelled result button.
